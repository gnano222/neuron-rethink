① input · 14×14

SPROUT never sees your full drawing. It's cropped, centered, shrunk to a 14×14 thumbnail and normalized to look like the MNIST digits it trained on. This is the actual input — if it looks wrong here, the model will struggle.

brighter = stronger pixel after standardization

② filters → feature maps

Each learned 3×3 filter (left) is slid across your image to produce its feature map (right) — a picture of where that pattern appears in your digit. Each filter has exactly one map. Brightly-lit pairs fired strongly; SPROUT learned and pruned the filters itself, so only the survivors are shown.

③ head network

The sparse classifier. It reads the feature maps (left column), combines them through hidden neurons (middle), and votes for a digit (right). Watch the active circuit light up — the specific path your drawing took through the network.
